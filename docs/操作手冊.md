# 📖 網站操作手冊

> cuteHsu 個人網站 — 日常維護與更新指南

---

## 🚀 快速部署（最常用）

每次修改完程式碼，只要執行這一行：

```bash
pnpm run deploy
```

它會自動完成：編譯 → 推送到 GitHub → 網站更新

---

## 🛠️ 開發環境設定（第一次 / 換電腦時）

### Step 1：確認 Node.js 版本

```bash
node --version
```

> ⚠️ 必須是 **v20 以上**，否則編譯會失敗

如果版本太低，用 nvm 切換：
```bash
nvm use 22
```

### Step 2：安裝套件

```bash
pnpm install
```

### Step 3：啟動本地預覽

```bash
pnpm run dev
```

打開瀏覽器輸入 `http://localhost:5174/mywebsit/` 即可預覽

## ✏️ 修改網站內容

### 修改首頁標題 / 副標題

開啟 `src/components/HelloWorld.vue`：

```vue
<h1 class="hero-name">{{ msg }}</h1>        <!-- 主標題，在 App.vue 傳入 -->
<p class="hero-subtitle">Enjoying life...</p> <!-- 副標題，直接改這裡的文字 -->
```

主標題在 `src/App.vue` 第 79 行傳入：
```vue
<HelloWorld msg="cuteHsu" />   ← 改這裡的文字
```

### 修改卡片連結

同樣在 `TheWelcome.vue`，把 `<WelcomeItem>` 改成可點擊的連結：


## 🎨 修改視覺樣式

開啟 `src/assets/base.css`，修改色彩變數：

```css
:root {
  --botw-cyan:  #4af9f9;   /* Sheikah 青色（邊框發光）*/
```

開啟 `src/App.vue`，修改陣列長度：

```js
// 星星數量（預設 90）
const stars = Array.from({ length: 90 }, ...)

// 精靈光點數量（預設 18）
const wisps = Array.from({ length: 18 }, ...)
```

## 📤 完整更新流程

```
1. 修改程式碼（src/ 資料夾內的 .vue 或 .css）
      ↓
2. 本地預覽確認（可選）
   pnpm run dev
      ↓
3. 部署上線
   pnpm run deploy
      ↓
4. 等待 1~3 分鐘
      ↓
5. 打開網站確認
   https://alenhsu1120.github.io/mywebsit/
```

## ❓ 常見問題

### Q：執行 `pnpm run deploy` 出現錯誤？

**A：** 先確認 Node 版本：
```bash
node --version   # 需要 v20+
nvm use 22       # 切換版本
pnpm run deploy  # 重新執行
```

---

### Q：網站更新了但瀏覽器還是舊的？

**A：** 強制重新整理清除快取：
- Windows / Linux：`Ctrl + Shift + R`
- Mac：`Cmd + Shift + R`

---

### Q：如何確認部署是否成功？

**A：** 打開 GitHub Actions 查看狀態：
👉 https://github.com/alenhsu1120/mywebsit/actions

看到 `pages build and deployment` 變成綠色 ✅ 就代表完成。

---

### Q：想回到改版前的舊版設計？

**A：** 用 git 還原：
```bash
git log --oneline          # 查看 commit 歷史
git checkout <舊的commit>  # 切換到舊版本
pnpm run deploy            # 重新部署
```
